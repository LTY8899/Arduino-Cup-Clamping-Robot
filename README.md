# Arduino Cup Clamping Robot

# Project Overview
This project is part of my Electronic Engineering course. The goal is to design a robot that can clamp and move as many cups as possible within a time limit, following a fixed path.

**Final Result: Clamped 13 cups in 2.5 minutes.**

# My Contribution
- Designed the robot's path planning strategy
- Added three sensors (left, center, right) to improve line-following accuracy
- Optimized the control logic and timing parameters in C++
- Collaborated with a teammate on hardware modifications

# Challenges & Solutions
- **Problem**: The initial clamp design would knock over cups.
- **Solution**: Redesigned the path and added sensors to keep the robot aligned.

# Result
- Achieved **Grade A** in the course
- **Clamped 13 cups in 2.5 minutes** under timed conditions

# Technologies Used
- C++
- Arduino
- State Machine (FSM)
- Sensors & Servo Motors
